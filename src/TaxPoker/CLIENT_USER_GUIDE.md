# 德州扑克客户端用户使用指南

## 📋 简介

欢迎使用德州扑克客户端！这是一个图形化的德州扑克游戏客户端，可以连接到本地服务器与其他玩家（包括AI机器人）一起游戏。

## ▶️ 快速开始

### 1. 启动服务器
```bash
cd Server
./server.out
```

### 2. 启动客户端
```bash
cd Client
python3 ClientMain.py
```

## 🎮 界面介绍

### 标题栏
- 显示应用程序名称"德州扑克游戏"

### 玩家信息区
- **布局**：上下两排，每排4个玩家位置
- **信息**：
  - 玩家名称
  - 当前筹码数量
  - 最近操作
  - 本轮下注金额

### 公共牌区
- **我的手牌**：显示您的两张私人牌
- **公共牌**：显示桌面上的公共牌（翻牌、转牌、河牌）

### 底池信息区
- **游戏阶段**：显示当前处于哪个阶段（翻牌前、翻牌后等）
- **底池金额**：显示当前底池中的总筹码数

### 操作按钮区
- **弃牌(Fold)**：放弃当前手牌
- **过牌(Check)**：不下注继续游戏（当无需跟注时可用）
- **跟注(Call)**：跟随当前最高下注额
- **加注(Raise)**：增加下注额（需要在输入框中输入金额）
- **全押(All-in)**：押上所有剩余筹码

### 系统信息区
- 显示连接状态、游戏进度、错误信息等

## 🎯 游戏操作说明

### 连接服务器
1. 确保服务器已在运行
2. 启动客户端程序
3. 客户端会自动连接到本地服务器(127.0.0.1:10002)
4. 连接成功后会显示玩家ID和游戏信息

### 游戏流程
1. **等待游戏开始**：系统会自动匹配玩家
2. **接收手牌**：您的两张私人牌会显示在"我的手牌"区域
3. **进行操作**：根据当前游戏状态，相应的操作按钮会被激活
4. **输入加注金额**：如果选择加注，需要在输入框中输入金额
5. **确认操作**：点击相应按钮完成操作

### 操作按钮详解

#### 弃牌(Fold)
- 放弃当前手牌，退出本轮游戏
- 点击红色"弃牌"按钮即可

#### 过牌(Check)
- 不下注继续游戏
- 仅在无需跟注时可用
- 点击蓝色"过牌"按钮

#### 跟注(Call)
- 跟随当前最高下注额
- 点击紫色"跟注"按钮

#### 加注(Raise)
1. 在输入框中输入想要加注的金额
2. 点击橙色"加注"按钮
3. 系统会验证金额是否有效

#### 全押(All-in)
- 押上所有剩余筹码
- 点击深橙色"全押"按钮

## ⚠️ 注意事项

### 筹码管理
- 初始筹码为500
- 所有下注都会从您的筹码中扣除
- 赢得底池会增加您的筹码

### 输入验证
- 加注金额必须为正整数
- 加注金额不能超过您的剩余筹码
- 输入无效金额时会显示错误提示

### 错误处理
- 网络连接失败时会显示错误信息
- 服务器断开连接时程序会提示
- 异常情况下程序会尝试优雅处理

## 🛠️ 常见问题

### Q: 连接服务器失败怎么办？
A: 请检查：
1. 服务器是否已启动
2. 服务器是否在监听10002端口
3. 防火墙是否阻止了连接

### Q: 界面显示异常怎么办？
A: 请检查：
1. Python和Tkinter是否正确安装
2. 系统是否有足够的显示空间
3. 是否设置了合适的显示分辨率

### Q: 如何退出游戏？
A: 关闭客户端窗口即可退出游戏
